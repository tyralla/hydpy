hstream_v1 (HBV96 version of HydPy-H-Stream)¶

class hydpy.models.hstream_v1.Model[source]¶

Bases: hydpy.core.modeltools.Model

The HBV96 version of HydPy-H-Stream (hstream_v1).

The following “run methods” are called each simulation step run in the given sequence:
  • calc_qjoints_v1() Apply the routing equation.
  • update_inlets_v1() Assign the actual value of the inlet sequence to the upper joint of the subreach upstream.
  • update_outlets_v1() Assing the actual value of the lower joint of of the subreach downstream to the outlet sequence.
